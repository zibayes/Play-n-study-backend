### Перед запуском предложения развернуть бд
1. Запустить postgres в docker / на локальной машине / сервере

2. Изменить app.config (9 строка app.py)
3. Прописать в директории проекта в терминале следующие команды

```$ flask db init```
   
   ```$ flask db migrate``` 

```$ flask db upgrade```
  
   для развертывания бд
